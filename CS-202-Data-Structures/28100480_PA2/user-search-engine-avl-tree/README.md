User Search Engine using AVL Trees

This project implements a user search system using Binary Search Trees
and self-balancing AVL Trees.

Key Features

• AVL tree implementation with automatic rebalancing
• Binary Search Tree operations including insert, delete, and search
• Efficient user indexing by ID and username
• Prefix-based username search
• Fuzzy username search using edit distance
• Range queries on user IDs

Concepts Demonstrated

• Self-balancing tree structures
• Tree rotations (LL, RR, LR, RL)
• Tree traversal algorithms
• Edit distance algorithm
• Efficient search indexing
