Social Network Algorithms Engine (C++)

This project implements core data structures and algorithms used in
modern social media platforms.

Key Components

• Hash table with linear probing, quadratic probing and separate chaining
• Min heap for ranking and feed prioritization
• Graph algorithms including Dijkstra and Kruskal
• User interaction graph with similarity analysis
• PageRank based social influence scoring
• Community detection using Kosaraju’s algorithm
• Recommendation system based on Jaccard similarity

Concepts Demonstrated

• Hashing and collision resolution
• Priority queues and heap structures
• Graph theory algorithms
• Social network analysis
• recommendation systems
